https://yaxuanpang.github.io/cart263/topics/project1/ 

I enjoyed this one. I’ve played games like it before and it’s fun. One thing I found about the gameplay I thought was a bit unfair was that sometimes squares would spawn on top of circles so I wouldn’t be able to collect the squares without losing a life. Also, I wished you could play it with the wasd keys instead of the arrow keys. 

I wasn’t able to win, part of me thought it was impossible to win until I saw the gameWin div. The files and everything all seem to be well orginized but the naming convention seems to be strange to me. There are no birds in the game, but they use the term “bird” for the circles.  

 

https://philippe-bn.github.io/cart263/projects/project-1/ 

This one was fun to interact with. I love the idea; it truly is like an aquarium.  

The code is well organized, and the commit messages are clear. I think they may have over complicated their project with the vector js file. They could’ve just drawn a fish and added some size variations for it. And if you were drawing them, it would be easy to add variation of fish. You could make an array with plenty of different types of fish.  The fish.checkEdges(); in function animateFish() was very thoughtful.  

 

https://norahwilcox.github.io/CART263/projects/project-1/Project1/ 

I liked this one. I enjoyed the aesthetic and idea. It reminds me of when I played with the photobooth app on a MacBook for the first time when I was 12. The Windows Vista wallpaper is very nostalgic.  

I found the file organization in the github repository slightly confusing to navigate at first. The true project is inside the “Project1” folder and not the “project-1” folder it was inside of even though there are pngs that belong in the projects in that folder. It’s especially confusing since no one linked their repository, I found it with their username in the url and used that to find them, then I navigated to their projects. It was easy once I figured it out. It was thoughtful to add a Safari fallback since it was having issues (according to the notes). I feel like most people would’ve just put a disclaimer to not use Safari.
